Backup & Restore

Move your loads between your devices — no account or setup needed. Export a backup file on one device, send it to the other (AirDrop, Messages, email, iCloud Drive, etc.), then Restore it there.

Move Data to Another iPhone/iPad

The easiest way is AirDrop:

  1. On this device, tap Export Backup above. Safari saves a file named load-tracker-backup-[date].json to your Downloads.
  2. Open the Files app → Downloads, find that file, then touch and hold it → Share → AirDrop → pick your other device (both need Wi-Fi/Bluetooth on and be nearby, unlocked).
  3. On the other device, tap Accept on the AirDrop notification — it saves to that device's Files → Downloads.
  4. If Load Tracker isn't set up on that device yet: AirDrop the Load Tracker.html file itself the same way, open it in Safari, then Share → Add to Home Screen.
  5. On that device, open Load Tracker → ⚙️ Settings tab → tap Restore from Backup → choose the file you just AirDropped. Your routes and loads merge in automatically.

No AirDrop / devices not nearby? When exporting, choose Save to Files → iCloud Drive instead of Downloads — the file (and the app file too) will then show up in Files on all your Apple devices signed into the same iCloud account.
