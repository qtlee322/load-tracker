Real-Time Cloud Sync
How to Set Up Real-Time Sync (one time)
  1. Go to console.firebase.google.com, sign in with a Google account, tap Add project, name it, skip Google Analytics, tap Create project.
  2. Build → Authentication → Get started → Sign-in method tab → enable Google → Save.
  3. Build → Firestore Database → Create database → pick a location → Start in production mode → Create. Then open the Rules tab, replace everything with the rules below, and tap Publish.
  4. Build → Storage → Get started → keep the defaults → Done. Then open its Rules tab too, paste the storage rules below, and Publish.
  5. Project settings (gear icon) → General, scroll to "Your apps", tap the </> (web) icon, give it any nickname, register it. It shows a code block starting with const firebaseConfig = {...} — copy that whole block.
  6. This app is already connected to Firebase — on any device, just tap Sign in with Google below with the same Google account to link that device in. Everything then syncs automatically and instantly.
